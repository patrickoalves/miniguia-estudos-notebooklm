# Caderno Temático com NotebookLM: Introdução à Ciência de Dados

Este projeto foi desenvolvido para explorar o uso da Inteligência Artificial como apoio ao estudo, à curadoria de fontes e à organização do conhecimento.

## Tema e objetivos

O tema escolhido foi **Introdução à Ciência de Dados**.

Objetivos:
- Entender os conceitos básicos da área.
- Identificar etapas, ferramentas e aplicações iniciais.
- Diferenciar ciência de dados de áreas próximas.
- Criar um material de revisão simples e reutilizável.

## Uso do NotebookLM

As fontes foram organizadas no NotebookLM para apoiar:
- Resumos automáticos.
- Explicação dos conceitos.
- Comparação de ideias próximas.
- Criação de material de revisão.

## Prompts testados

- O que é ciência de dados?
- Quais são os pilares da ciência de dados?
- Qual a diferença entre ciência de dados, estatística e machine learning?
- Crie um resumo em tópicos sobre introdução à ciência de dados.
- Monte um glossário com os principais conceitos do tema.

## Dificuldades encontradas

- Algumas respostas foram muito genéricas.
- Foi necessário refinar os prompts para obter respostas mais úteis.
- Perguntas mais específicas geraram resultados melhores e detalhados.

## Miniguia de estudo

### Resumo rápido
A ciência de dados é uma área interdisciplinar que usa estatística, programação e análise para extrair conhecimento de dados e apoiar decisões.

### Glossário
- **Dado:** registro bruto de uma observação.
- **Informação:** dado interpretado.
- **Insight:** conclusão útil extraída da análise.
- **Modelo:** representação usada para explicar ou prever algo.
- **Machine learning:** técnica que permite aprender padrões a partir de dados.

### Prompts reutilizáveis
- Explique este tema em linguagem simples.
- Resuma os conceitos mais importantes em tópicos.
- Compare ciência de dados, estatística e machine learning.
- Crie um glossário com os termos principais.
- Liste aplicações práticas da ciência de dados.

## Conclusão

O projeto ajudou a organizar o estudo sobre ciência de dados de forma clara e objetiva, mostrando como a IA pode apoiar a aprendizagem ativa.
